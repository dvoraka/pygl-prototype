Introduction
==============

Python OpenGL prototype. It has prepared structure for future development.

### Controls

Key | Action
---|---
UP | forward
DOWN | backward
LEFT | step left
RIGHT | step right
Page Up | go up
Page Down | go down
L | lines rendering
F | full rendering
